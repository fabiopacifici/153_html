Note:
Inside `ul` or `ol` tags you MUST only add `li` as first level childs

❌ Error!

```html


    <ul>
      <li></li>
      <li></li>
      <li></li>
      <strong></strong>
      <p></p>
    </ul>


```

✅ Correct!

```html


    <ul>
      <li></li>
      <li></li>
      <li>
        <strong></strong>
        <p></p>
      </li>
      
    </ul>


```
